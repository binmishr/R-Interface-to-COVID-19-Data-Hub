Built with R, available in any language, COVID-19 Data Hub provides a worldwide, fine-grained, unified dataset helpful for a better understanding of COVID-19. The user can instantly download up-to-date, structured, historical daily data across several official sources. The data are hourly crunched and made available in csv format on a cloud storage, so to be easily accessible from Excel, R, Pythonâ€¦ and any other software. All sources are properly documented, along with their citation.
